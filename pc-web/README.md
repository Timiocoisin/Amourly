# 择爱 - PC端网页

## 技术栈
- Vue 3
- TypeScript
- Vite
- Vue Router
- Pinia
- Sass

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 项目结构

```
src/
  components/     # 组件
  views/          # 页面
  router/         # 路由配置
  store/          # 状态管理
  styles/         # 全局样式
  utils/          # 工具函数
  api/            # API 接口
  types/          # TypeScript 类型定义
```
